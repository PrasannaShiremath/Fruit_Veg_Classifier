# 🥗 AI Fruit & Vegetable Classifier

An AI-powered web application that identifies 36 types of fruits and vegetables using deep learning.

## 🚀 Features
- Real-time image classification
- Confidence score display
- Simple UI built with Streamlit

## 🛠️ Tech Stack
- TensorFlow/Keras
- Streamlit
- Python
- CNN (Convolutional Neural Network)

## 📊 Model
- Input: 180x180 RGB images
- Classes: 36 categories

## 💻 Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
